# SQL-Fundamentals
This exercise introduces the fundamental concepts of SQL by focusing on retrieving and filtering data from a single employees table. It provides practical experience with essential SQL commands such as `SELECT`, `DISTINCT`, `WHERE`, `ORDER BY`, and `LIMIT`, helping learners understand how to extract specific information, sort results, remove duplicates, and apply conditions to narrow down data. The exercise also develops the ability to predict query outputs by requiring students to draw the expected result tables for each query. 

The second part of the exercise strengthens knowledge of filtering using conditional logic, including operators such as `AND`, `OR`, `NOT`, and `IN`. Through ten handwritten questions, learners practice writing correct SQL syntax while improving logical thinking and understanding of how databases respond to queries. Overall, the exercise builds a strong foundation in SQL fundamentals and prepares students for more advanced database querying concepts.
